# RedEye VB — Support

This repository hosts the marketing site, support documentation, and the public bug tracker for **RedEye VB**, the Nintendo Virtual Boy emulator for Mac.

The emulator's source code is **not** in this repository.

- **Site:** https://remmylee.github.io/redeye-vb-support/
- **Support:** https://remmylee.github.io/redeye-vb-support/support.html
- **Privacy Policy:** https://remmylee.github.io/redeye-vb-support/privacy-policy.html
- **Report a bug or request a feature:** [Open a new issue](../../issues/new/choose)

## Why is this repo public?

So you can:

1. Read the user-facing site (rendered via GitHub Pages)
2. Open issues for bugs, feature requests, and questions

If you don't have a GitHub account, [create one for free](https://github.com/join) — it takes about a minute.
